# Mix Modeler チュートリアル

これらのビデオとチュートリアルから、Adobe Mix Modelerについて説明します。

Adobe Mix Modelerは、主要なビジネス目標に対するマーケティング投資の過去および将来の影響に関するインサイトを提供する機械学習モデルを適用することで、企業がマーケティング投資を測定、最適化、計画するのに役立ちます。
