# Mix Modeler チュートリアル

これらのビデオとチュートリアルからAdobe Mix Modelerについて説明します。

Adobe Mix Modelerは、マシンラーニングモデルを適用し、マーケティングへの投資が主要なビジネス目標に与えた効果を過去および将来にわたって明らかにすることで、企業がマーケティングへの投資を測定、最適化、計画するのを支援します。
