# 目標ベースの計画

Mix Modelerの目標ベースの計画機能を使用すると、予算だけでなくビジネス目標から計画できます。 最大の ROI で目標を達成するために、チャネルやキャンペーンをまたいで効率的に予算を割り当てることができます。
